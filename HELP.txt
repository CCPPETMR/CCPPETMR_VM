To make interfaces to Gadgetron and STIR forks on CCPPETMR up-to-date, 
type update_VM.sh.

To run Gadgetron interface demos, first open a new terminal and type gadgetron.
Then open another terminal, change to ~/devel/xGadgetron/examples and type 
python demo1.py etc. Type demo1.py -h to get a brief description of demo1.py
and its usage.

To run STIR Python interface demos, change to ~/devel/xSTIR/examples and type 
python stir_demo1.py etc. 

If you want to stay abreast with the latest developments:

Change to ~/devel/CCPPETMR and type git pull. 

Change to folder 'scripts' and type chmod +x *.sh, then type ./ADD_REMOTE.sh. 
This is only needs to be done ONCE.

Whenever you want to update your VM:

- Type export TMP=<password you used to log in>.

- Type ./UPDATE_ALL.sh.

- When the above script finished (it may take about 1 hour), you may like to 
  erase the value of TMP by typing export TMP=
