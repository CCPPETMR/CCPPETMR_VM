To make your Virtual Machine up-to-date, type update_VM.sh.

To run Gadgetron interface demos, first open a new terminal and type gadgetron.

To run Gadgetron Python interface demos, open another terminal, change to 
~/devel/xGadgetron/examples and type python demo1.py etc. To run Matlab demos,
change Matlab work folder to ~/devel/xGadgetron/examples and type demo1 etc.

To run STIR Python interface demos, change to ~/devel/xSTIR/examples and type 
python stir_demo1.py etc. To run Matlab demos, change to this folder in Matlab
and type stir_demo_1 etc.
