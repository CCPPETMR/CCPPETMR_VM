# Building CCPPETMR_VM via Vagrant

## Installation

- Install VirtualBox
- Install [Vagrant](https://www.vagrantup.com)

## Vagrant commands

- Start the machine

```
cd vagrant
vagrant up
```

- Pause the machine

```
vagrant suspend
```

- Shutdown the machine

```
vagrant halt
```

- Remove the machine (to start from scratch)

```
vagrant destroy
```
